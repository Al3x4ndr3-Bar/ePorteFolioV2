Outils à ma disposition

Voici le matériel dont je dispose pour ma formation actuel.

    Idle :

    IDLE est l'environnement de développement intégré officiel de Python.
    Il est conçu pour faciliter l'écriture, l'exécution et le test de scripts Python.
    L'interface est simple et intuitive, rendant l'apprentissage du langage Python plus accessible. De plus, avec son shell interactif, IDLE permet d'exécuter des commandes Python en temps réel.
    Télécharger Python (inclut IDLE).

    Chat-GPT :

    Chat-GPT est une évolution de l'algorithme GPT optimisée pour des interactions conversationnelles.
    Cet outil est précieux pour améliorer la communication avec les modèles d'intelligence artificielle, en particulier pour obtenir des informations ou des conseils sur la programmation.
    Visiter OpenAI.

    Symfony :

    Symfony est un framework PHP dédié au développement web.
    Il offre une structure solide pour construire des applications web performantes et bien organisées.
    Grâce à son système de "bundles", Symfony facilite la modularité et l'ajout de fonctionnalités à des projets.
    Télécharger Symfony.

    WampServer :

    WampServer est une solution complète pour le développement web sous Windows.
    Il intègre des outils essentiels comme Apache2, PHP et MySQL, permettant ainsi de développer, exécuter et tester des applications web en local.
    Télécharger WampServer.

    Visual Studio Code :

    Visual Studio Code, développé par Microsoft, est un éditeur de code polyvalent qui supporte une multitude de langages.
    Sa force réside dans son écosystème d'extensions qui étendent ses fonctionnalités, le rendant adaptable à de nombreux scénarios de développement.
    Télécharger Visual Studio Code.

    Visual Studio 2022 :

    Visual Studio 2022 est un environnement de développement intégré robuste produit par Microsoft.
    Conçu pour plusieurs langages de programmation, il dispose d'outils avancés pour concevoir, déboguer et tester des applications d'envergure.
    Télécharger Visual Studio 2022.
